# 1k PBR Lighting Test

This is an interactive WebXR demo built with A-Frame, showcasing Physically Based Rendering (PBR) techniques in a 3D environment. The project demonstrates advanced lighting and material rendering capabilities directly in the browser.

## Overview

The demo features a 3D scene with PBR materials and lighting effects, optimized for web delivery. It includes real-time performance monitoring and supports immersive VR experiences through WebXR.

## Key Features

- **WebXR Integration**: Enter VR mode with a headset or experience fullscreen without one
- **PBR Rendering**: Physically based materials and lighting for realistic visuals
- **Performance Stats**: Real-time display of framerate, memory usage, render calls, and geometry counts
- **Cross-Platform**: Works across different devices and browsers supporting WebXR

## Technical Details

- **Framework**: A-Frame (built on Three.js)
- **Rendering**: WebGL with PBR shaders
- **Textures**: High-resolution texture support (8k mentioned in title)
- **Entities**: 37 A-Frame entities in the scene
- **Performance**: Optimized for smooth 60fps rendering
